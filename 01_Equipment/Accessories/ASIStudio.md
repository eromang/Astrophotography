# ASIStudio

> **Installed version:** 1.18
> **Location:** `/Applications/ASIStudio.app`

ZWO's desktop software suite for camera control, capture, stacking, and mount management.

https://www.zwoastro.com/software/

---

## Bundled Applications

| Application | Version | Purpose |
|-------------|---------|---------|
| ASICap | 2.18 | Camera capture (images and video) |
| ASILive | 1.18 | Live stacking and EAA (Electronically Assisted Astronomy) |
| ASIDeepStack | 1.18 | Deep sky image stacking |
| ASIImg | 1.18 | Image viewer and basic processing |
| ASIFitsView | 1.18 | FITS file viewer |
| ASIVideoStack | 1.18 | Planetary / lunar video stacking |
| ASIMeteorCap | 1.18 | Meteor detection and capture |
| ASIMountController | 1.18 | Mount control (GoTo, tracking) |
| ASISDKTool | 1.18 | SDK diagnostic tool |
| ChannelCombine | — | Combine mono channels into RGB |
| FormatConverter | — | Image format conversion |

## Relevant to Setup

| Application | Use Case |
|-------------|----------|
| ASICap | Direct camera control for [[ASI2600MCPro]] and [[ASI385MC]] when not using [[ASIAIR]] |
| ASIVideoStack | Planetary imaging with [[Celestron-NexStar-90SLT]] (lucky imaging) |
| ASIMountController | Direct mount control for [[iOptron-CEM26]] |
| ASIDeepStack | Quick stacking alternative to PixInsight WBPP |
| ASIFitsView | Quick inspection of FITS files before processing |
