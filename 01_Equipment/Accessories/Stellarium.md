# Stellarium

> **Installed version:** 25.4
> **Location:** `/Applications/Stellarium.app`

Open-source planetarium software for session planning and sky visualization.

https://stellarium.org/

---

## Use Cases

| Use Case | Description |
|----------|-------------|
| Session planning | Visualize target positions, altitude, and transit times for a given date |
| FOV simulation | Overlay [[RedCat-51]] + [[ASI2600MCPro]] field of view on targets |
| Moon avoidance | Check moon position and proximity to targets |
| Object identification | Identify objects visible from observing location |
| Ephemerides | Sunset, sunrise, twilight times for Tuntange, Luxembourg |

---

## Configuration

- **Location:** Tuntange, Luxembourg (49.6°N, 6.1°E, 310m altitude)
- **FOV sensor:** ASI2600MC Pro (23.5 x 15.7mm) + RedCat 51 (250mm f/4.9)
  - Resulting FOV: ~5.4° x 3.6°
