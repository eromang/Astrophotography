# Optolong L-eNhance Filter 1.25"

Dual-narrowband filter passing Ha and OIII/Hb emission lines. Designed for use with the [[Celestron-NexStar-90SLT]] (1.25" eyepiece holder) for visual observation and short-exposure imaging of emission nebulae.

- [Optolong product page](https://www.optolong.com/cms/document/detail/id/11.html)

---

## Specifications

| Specification | Value |
|---------------|-------|
| Type | Dual narrowband (multi-bandpass) |
| Bandpass 1 | H-alpha (Ha) — 656nm |
| Bandpass 2 | H-beta (Hb) — 486nm |
| Bandpass 3 | Oxygen III (OIII) — 496/500nm |
| Bandpass width | ~10nm (Ha), ~24nm (OIII/Hb) |
| Size | 1.25" (31.75mm) |

---

## When to Use

Used with the [[Celestron-NexStar-90SLT]] for:
- Visual observation of emission nebulae (enhances contrast on Ha and OIII targets)
- Short-exposure imaging of bright emission objects
